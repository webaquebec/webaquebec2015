Atelier (sur inscription) : Introduction au machine learning avec scikit-learn

Mer 18 10h 25 Salle Bell · Keynotes et ateliers

Note importante : les places pour participer aux ateliers du WAQ sont limitées et vous devez vous inscrire à l’avance. Si vous n’avez pas reçu de lien pour vous inscrire après l’achat de votre billet du WAQ, communiquez avec info@webaquebec.org pour en obtenir un.

Cet atelier est une introduction à quelques concepts de base en «machine learning».

Nous nous attarderons à l’apprentissage supervisé et nous verrons entre autres la préparation des données, la création d’un modèle ainsi qu’une méthode de validation de ce dernier.

La librairie Python qui très facile à s’approprier, scikit-learn, sera utilisée pendant l’atelier. Nous aurons l’occasion de tester divers algorithmes d’apprentissage sur un jeu de données.
